1. Push repository to GitHub.
2. Open [vercel.com](https://vercel.com) -> New Project -> import repository.
3. Add the 3 environment variables in Vercel.
4. Deploy.
5. Copy the Vercel URL.
6. Update `NEXT_PUBLIC_SITE_URL` in Vercel env to that URL.
7. Add Vercel URL to Supabase Auth -> URL Configuration -> Site URL.
8. Add Vercel URL to Google OAuth authorized origins.
9. Redeploy.
